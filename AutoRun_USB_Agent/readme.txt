Программа No AutoRun_USB Agent предназначена для ликвидации угроз распространения
вирусов распространяющихся через autorun.inf файлы. Программа загружается
в оперативную память и при подключении новых дисков автоматически
переименовывает находящиеся на них файлы autorun.inf (при их наличии)
в autorun.inf_detect (делая их при этом видимыми), в результате чего 
значительно снижается вероятность заражения компьютера вирусами,
распространяющимися через мобильные носители информации.
